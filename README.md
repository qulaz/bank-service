# Bank Service

Сервис онлайн-банкинга с REST API для управления счетами и транзакциями.

## Требования

- Go 1.16+
- SQLite3

## Установка и запуск

1. Клонируйте репозиторий:

2. Установите зависимости:
```bash
go mod tidy
```

3. Запустите сервер:
```bash
go run cmd/server/main.go
```

Сервер будет доступен по адресу: http://localhost:8080

## API документация

Swagger документация доступна по адресу: http://localhost:8080/swagger/index.htm

## Основные функции

1. **Управление счетами**
   - Создание нового счета
   - Просмотр списка счетов
   - Просмотр детальной информации о счете

2. **Транзакции**
   - Перевод средств между счетами
   - Просмотр истории транзакций по счету

3. **OTP (One-Time Password)**
   - Генерация OTP-кода для подтверждения операций
   - Проверка OTP-кода

## Задание

В сервисе есть несколько проблем, которые необходимо найти и исправить:

1. Некоторые пользователи жалуются, что иногда не могут подтвердить операции с помощью OTP-кода. Необходимо выяснить причину и исправить проблему.

2. При одновременных переводах с одного счета иногда возникают проблемы с балансом. Необходимо обеспечить корректную работу с балансом при параллельных операциях.
